Today29Feb API Docs.

Main info:

Today29Feb is my joke api that will tell you whether today is February 29 or not, and also determine how many days are left until February 29.




Usage API:

Base URL for all endpoints: http://today29feb.ml
PathDescriptionParametrs
/apiReturns information about is today 29 feb. or no and determine how many days are left until for it No


Example for usage API on in different programming languages:


• Python:

import requests
import json

response = requests.get('http://today29feb.ml/api/')
json_data = json.loads(response.text)
print(json_data['today_29_feb'] + ' ' + json_data['when_29_feb'])



• JavaScript:

const fetch = require('node-fetch')

fetch('http://today29feb.ml/api/')
.then(res => res.json())
.then(json => {

console.log(json.today_29_feb + ' ' + json.when_29_feb)

});


Result: No. After 718 days
